1/ cài đặt project:
 npx create-react-app ten-app
2/ Chạy project
  npm run start
3/ flow code của ReactJS
 index.html ( chú ý id "root")
=>index.js(chú ý <App />)
=>App.js(chú ý return của function App)
4/ ReactJS không đọc dc html, mà chỉ đọc được jsx
5/ JSX giúp cho chúng ta viết html và js vào một file duy nhất
6/ lợi ích và tiêu chí để chia component:
    - tái sử dụng lại ( ưu tiên )
    - dễ quản lý và mở rộng úng dụng
7/ có 2 dạng component là class và function, thường baayg dùng function nhìu hơn(công nghệ mới)
